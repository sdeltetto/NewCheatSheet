CSS Selectors Cheatsheet

A useful exercise

Selector Name Description
div Tag Selector Selects the element with the div tag
.red Class Selector Selects the elements with the class .red
#nav Id Selector Selects the element with the nav Id
div.row Selector Chaining Selects all elements with the div tag and the row class
[aria-hidden="true"] Attribute Selector Selects all elements with the aria-hidden attribute with a value of “true”
li a Descendent combinator Selects all a tags that are a child of li tags
li, a Group Selector Selects all a elements and all li elements
